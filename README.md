Cyber Security Study Group - Repository
Este repositório é o ponto central de armazenamento e organização de ativos técnicos do grupo de estudos. O objetivo é a consolidação de conhecimento prático e teórico, evitando a dispersão de informações e o acúmulo de materiais redundantes.

📂 Estrutura de Diretórios
/Documentação: Metodologias, frameworks de GRC (Governança, Riscos e Compliance) e resumos técnicos validados.

/Laboratórios: Scripts de automação, arquivos de configuração para ambientes controlados e logs de atividades (Red Team/Blue Team).

/Hardware_Hacking: Documentação de projetos envolvendo microcontroladores (ESP32, Arduino) e análise de vulnerabilidades em hardware.

/Certificações: Materiais de apoio rigorosamente selecionados para exames (Ex: CompTIA Security+).

🛠 Diretrizes de Contribuição
Qualidade sobre Quantidade: Não utilize este espaço para armazenar "PDFs esquecidos". Apenas materiais com aplicabilidade prática comprovada devem ser mantidos.

Segurança da Informação: É terminantemente proibido o upload de credenciais reais, chaves de API, segredos ou dados sensíveis. Utilize arquivos .env.example ou técnicas de masking.

Documentação: Todo script ou ferramenta adicionada deve vir acompanhada de uma breve descrição funcional no respectivo diretório.

Versionamento: Mantenha commits claros e objetivos. Alterações sem justificativa técnica serão revertidas.

Estado atual: Em desenvolvimento e curadoria constante.
